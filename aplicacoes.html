Projeto Hidrelétrica

O Projeto Hidrelétrica é um projeto escolar para a feira de ciências do SesiSenai, conhecida como Amostra STEAM. O projeto foi realizado pelo Erick Shuck, Giovanni Burgo, João F. Ventura, João V. Galiotto, Lucas Pereira e Nícollas Lopes, sendo instruido pelos professores Carlos Roberto, Wendel Souza e Eduardo Michereff.

Ao lado, é possível observar as redes sociais da instituição do Senai Tijucas, além do GitHub do projeto, ao qual é possível visualizar código utilizado na construção do site em que você estálendo.
